# Multilingual-ASR-Benchmarking-Fine-Tuning-for-Real-World-Hindi-Speech-Using-Whisper
Built an end-to-end ASR pipeline to evaluate and improve Whisper on real-world Hindi speech. Conducted baseline benchmarking, noise robustness testing, and domain fine-tuning using Common Voice. Achieved measurable WER/CER improvements and performed error analysis across clean and noisy conditions.
